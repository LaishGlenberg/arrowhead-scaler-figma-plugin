# Arrowhead Scaler Figma Plugin

**Built by**: LaishGlenberg

**Description**: Plugin for making custom arrow heads and using more traditional, anchored scaling style for arrow heads. Makes stroke thickness updates applied consistently across the arrow. Especially useful with triangle arrow heads/ends or circular arrows where traditional figma stroke thickness breaks.

**How it works**: This plugin works by removing the normal start/end point arrow heads and replacing them with vector shapes that can be controlled with custom dimensions and scaling. Unlike before, each arrowhead line can now be arranged assymetrically allowing for far greater control over arrow designs. The only downside is that you can't use Figma's native start/end point arrowheads while the plugin is enabled on an arrow

**Status**: This is a work in progress and the plugin is still in development.
